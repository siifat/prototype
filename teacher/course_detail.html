Course Description

This course provides a comprehensive introduction to database management systems. Topics include data modeling, relational database design, SQL programming, normalization, transaction processing, and database administration. Students will gain hands-on experience with modern database systems.

Learning Outcomes

  • Design and implement relational databases using ER modeling
  • Write complex SQL queries for data manipulation and retrieval
  • Apply normalization techniques to optimize database schemas
  • Understand transaction management and concurrency control
